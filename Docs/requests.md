## 2026-05-10 17:40:59 (소요시간: 11분 10초)

- Enhanced Input 기반 탑다운 캐릭터 이동을 구현할 것.
- 에디터 모듈로 `IMC_`, `IA_` 입력 asset을 생성할 것.
- WASD 이동, 마우스 커서 방향 플레이어 회전, LMB 홀드 연사, RMB 홀드 조준 상태를 구현할 것.
- 조준 중 카메라 FOV를 좁히고 사선 방향으로 카메라 중심을 보정하되, 카메라는 회전하지 않게 할 것.
- 플레이어 BP는 임시 Static Mesh Cylinder로 만들고, 추후 Skeletal Mesh 연결을 고려할 것.
- 투사체 BP와 무기 BP를 만들고 플레이어에 무기를 attach할 것.
- 잔탄, 피해량 같은 게임 로직 없이 이동, 조준, 발사, 연발만 구현할 것.
- 적 BP를 만들고 AI는 AIController C++ 코드로 동작하게 하며, Behavior Tree는 프로젝트에서 사용하지 않을 것.
- Behavior Tree 미사용 방침을 `README.md`에 기록할 것.
