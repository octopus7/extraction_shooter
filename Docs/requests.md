- WASD 이동, 마우스 커서 방향 플레이어 회전, LMB 홀드 연사, RMB 홀드 조준 상태를 구현할 것.
- 조준 중 카메라 FOV를 좁히고 사선 방향으로 카메라 중심을 보정하되, 카메라는 회전하지 않게 할 것.

- 투사체 BP와 무기 BP를 만들고 플레이어에 무기를 attach할 것.
- 잔탄, 피해량 같은 게임 로직 없이 이동, 조준, 발사, 연발만 구현할 것.
- 적 BP를 만들고 AI는 AIController C++ 코드로 동작하게 하며, Behavior Tree는 프로젝트에서 사용하지 않을 것.
- Behavior Tree 미사용 방침을 `README.md`에 기록할 것.

## 2026-05-11 16:08:03 (소요시간: 25분 39초)

- 상호작용 표시 이름이 `열기`인 상자 액터를 구현하고, 상자 루팅정보 id를 액터에 지정할 수 있게 할 것.
- 상자 내용물은 별도 테이블에서 얻되 아직은 확정 정보만 표기하고, 범위나 확률은 나중에 구현할 것.
- 상자 외양과 최대 용량은 상자 정보 테이블에서 관리한 뒤 액터에 지정할 것.
- 게임 내 데이터 테이블은 하나의 서브시스템에서 로딩과 호출을 관리하고, 실제 테이블 클래스는 필요하면 따로 분리해도 무방함.
- 스폰 테스트용 액터처럼 `상자스폰` 상호작용 액터를 만들고, 런타임에 임의 상자 모양과 상자 내용물 정보 id를 지정해 상자를 스폰하게 할 것.
- 상자 용량은 5배수로 관리하고 현재는 5, 10, 15만 두되 나중에 확장 가능하게 할 것.
- 상자 내용물은 지정된 상자 용량을 초과하지 않게 하고, 내용물 가지수와 수량은 확정 데이터로 둘 것.
