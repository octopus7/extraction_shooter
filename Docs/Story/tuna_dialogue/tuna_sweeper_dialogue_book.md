# Tuna Sweeper 대화집

숲속 벙커에서 깨어난 메이드로이드 **루나**가,
작은 생활권을 다시 집으로 바꾸어 가는 본편 스토리 대화집입니다.

---

## 주요 화자

- **루나**: 플레이어 캐릭터. 차분하고 담담한 메이드로이드.
- **관리 AI**: 벙커 시스템. 건조한 문장으로 목표와 상태를 전달함.
- **캔봇**: 참치 선반과 보급품 정리를 담당하는 소형 로봇.
