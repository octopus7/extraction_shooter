# 3,000원 / 4시간 완결형 스팀 인디 게임 기획 정리

## 1. 가격과 플레이타임 기준

### 기본 가격 기준

- 한국 판매가 후보: **₩3,000**
- 목표 1회차 플레이타임: **약 4시간**
- 시간당 비용:
  - `₩3,000 ÷ 4시간 = 약 ₩750/h`

### 체감 가성비

| 기준 | 시간당 비용 | 평가 |
|---|---:|---|
| 매우 가성비 좋음 | ₩500/h 이하 | 강한 가성비 |
| 만족스러운 구매 | ₩500~₩1,000/h | 적정 |
| 보통 허용 가능 | ₩1,000~₩2,000/h | 장르/완성도 의존 |
| 비싼 편 | ₩3,000/h 이상 | 짧은 고퀄 작품일 때만 가능 |

**결론:** 4시간 플레이에 ₩3,000이면 가격 대비 플레이타임은 충분히 방어 가능하다.

---

## 2. 미국 달러 가격 감각

### 환율 기준 감각

- ₩3,000은 대략 **$2.1~$2.3** 정도의 환산가.
- 하지만 미국 유저 체감상 **$2.99도 여전히 저가 인디 가격대**로 받아들여질 가능성이 높다.

### 추천 가격 포지션

| 한국가 | 미국가 후보 | 느낌 |
|---:|---:|---|
| ₩3,000 | $1.99 | 매우 저렴한 미니게임 느낌 |
| ₩3,000 | $2.99 | 4시간 볼륨이면 자연스러운 저가 인디 |

**추천:** 4시간 플레이타임을 확실히 제공할 수 있다면 **₩3,000 / $2.99** 조합이 가장 깔끔하다.

### 출시 할인 전략

- 기본가: **$2.99**
- 출시 할인: **10~20%**
- 할인 체감가: 약 **$2.39~$2.69**

이렇게 하면 $2.99의 가격 방어력과 초기 진입장벽 완화를 동시에 얻을 수 있다.

## 3. Steam 국가별 가격 설정 방식

Steam은 Apple처럼 고정 티어 강제 방식이 아니라, **국가/통화별 가격을 직접 입력하는 방식**에 가깝다.

### 핵심

- USD 기준가를 입력하면 Steam이 국가별 권장가를 제안한다.
- 개발자는 권장가를 그대로 쓰거나, 일부만 수정하거나, 직접 국가별 가격을 조정할 수 있다.
- 즉, 한국은 **₩3,000**, 미국은 **$2.99**처럼 별도 설정 가능하다.

## 4. 본편 구조: 1지역 4시간 완결형

### 방향성

- 본편은 **1지역 사건을 완결**한다.
- 새 지역이나 DLC는 **떡밥/세계관 확장 암시** 정도로만 남긴다.
- 절대 “진짜 엔딩은 DLC에서”처럼 보이면 안 된다.

### 좋은 구조

> 본편: 이 지역에서 벌어진 문제는 해결했다.  
> 엔딩: 그런데 이 현상은 여기만의 문제가 아니었다.

### 엔딩 후 떡밥 예시

- 지도 밖 봉쇄된 새 지역
- 짧은 무전 신호
- 다른 지역 코드명 문서
- “ZONE-02: SIGNAL DETECTED” 같은 쿠키성 암시

**결론:** 끊어팔기가 아니라, 1지역은 완결하고 세계관은 열어두는 구조가 적합하다.

## 5. 전체 콘텐츠 볼륨 기준

### 목표

- 1회차 기준 약 4시간
- 업적 파밍, 2회차, 수집작은 플레이타임 계산에서 제외
- 본편 1회차만으로 가격 방어가 가능해야 한다.

### 권장 진행 구조

| 구간 | 플레이 감각 |
|---|---|
| 0~30분 | 튜토리얼, 첫 무기, 첫 귀환 |
| 30~90분 | 통신/전력/키카드 목표로 지역 탐색 |
| 90~150분 | 위험구역 개방, 샷건/소총 획득 |
| 150~210분 | 원인 추적, 보스 준비, 핵심 파밍 |
| 210~240분 | 최종 구역, 보스, 엔딩 |

## 6. 무기 파밍 볼륨

### 기본 방향

4시간 게임에서는 무기 수를 과하게 늘리기보다, **적당히 적고 역할 차이가 확실한 무기 구성**이 좋다.

### 최종 추천 구성: 총 10종

| 번호 | 무기군 | 역할 |
|---:|---|---|
| 1 | 근접무기 | 탄약 없을 때 쓰는 기본 생존용 |
| 2 | 기본 권총 | 초반 표준 무기, 탄약 효율 좋음 |
| 3 | 고위력 권총 | 느리지만 한 방 강함 |
| 4 | 기본 SMG | 근거리 연사, 잡몹 처리 |
| 5 | 고성능 SMG | DPS 높지만 탄약 소모 심함 |
| 6 | 기본 샷건 | 근접 폭딜, 탄약 희소 |
| 7 | 중거리 소총 | 정확도 좋고 안정적인 주력 |
| 8 | 고화력 소총 | 반동/탄소모 크지만 후반 주력 |
| 9 | 특수무기 | 보스/엘리트 대응용 |
| 10 | 네임드 무기 | 퀘스트/히든 보상용 변형 무기 |

### 소총 2종 역할 분리

| 소총 | 포지션 |
|---|---|
| 표준 소총 | 정확도 높음, 탄약 효율 좋음, 대부분 상황에서 안정적 |
| 전투 소총 | 데미지 높음, 반동 큼, 후반/엘리트몹 상대용 |

**결론:** 무기 20종보다, 10종 안팎에서 각 무기의 쓰임새를 확실히 나누는 편이 좋다.

## 7. 크래프팅과 설계도 시스템

### 판단

4시간 완결형에서는 **설계도 해금 + 제작 시스템은 오버스펙**일 가능성이 높다.

### 추천

| 요소 | 방향 |
|---|---|
| 설계도 시스템 | 제거 |
| 무기 제작 | 제거 |
| 제작대 | 제거하거나 최소화 |
| 무기 파밍 | 핵심으로 유지 |
| 부품 파밍 | 단순하게 유지 |
| 소모품 제작 | 필요하면 1~2개만 제한적으로 |

### 이유

플레이어가 기대하는 흐름은 다음에 가깝다.

> 탐색 → 파밍 → 장비 교체 → 퀘스트 진행 → 보스/엔딩

반면 설계도와 제작을 넣으면 다음 흐름이 추가된다.

> 재료 모으기 → 설계도 해금 → 제작대 복귀 → 제작식 확인 → 제작

4시간 게임에서는 이 과정이 템포를 늘어뜨릴 수 있다.

**결론:** 본편 1지역에서는 제작보다 파밍 손맛에 집중하는 편이 낫다.

## 8. 모딩/부품 시스템 방향

### 성격 정의

이 게임의 부품 시스템은 “입맛대로 커스터마이징하는 모딩”이 아니라, **언락형 장비 복구/강화**에 가깝다.

### 플레이어 감각

> “내 취향대로 총을 빌드한다”가 아니라  
> “망가진/부족한 장비를 파밍으로 점점 완성한다.”

### 추천 표현

| 표현 | 느낌 |
|---|---|
| 부품 장착 | 가장 무난 |
| 장비 보강 | 성장감 있음 |
| 무기 정비 | 세계관에 잘 맞음 |
| 결손 부품 복구 | 파밍 목적이 명확함 |
| 강화 부품 | RPG식 성장감 |
| 모듈 장착 | 약간 SF 느낌 |

### 구조

| 단계 | 상태 |
|---|---|
| 기본 획득 | 쓸 수는 있지만 불편함 |
| 부품 1개 장착 | 단점 하나 해소 |
| 부품 2개 장착 | 주력 무기로 쓸 만해짐 |
| 핵심 부품 획득 | 완성형 느낌 |

### 예시: 낡은 SMG

| 상태 | 효과 |
|---|---|
| 기본 | 반동 큼, 장탄수 적음 |
| 확장탄창 획득 | 지속 사격 가능 |
| 손잡이 획득 | 반동 감소 |
| 보정기 획득 | 완성형 느낌 |

### 피해야 할 방향

- 같은 슬롯에서 여러 부품 중 선택하는 복잡한 빌드트리
- 총구/탄창/손잡이별 다중 선택지
- 취향별 세팅 최적화
- RPG식 빌드 연구

**결론:** 부품은 선택지가 아니라 진행 보상으로 설계한다.

## 9. 퀘스트 시스템 구현 수준

### 방향성

4시간 1회차 기준에서는 복잡한 분기형 RPG가 아니라, **지역 탐색을 자연스럽게 밀어주는 체크리스트형 퀘스트**가 적합하다.

### 추천 구현 수준

| 요소 | 추천 |
|---|---|
| 메인 퀘스트 | 필수 구현 |
| 서브 퀘스트 | 소량 구현 |
| 히든 미니 퀘스트 | 선택적으로 구현 |
| 일일/반복 퀘스트 | 제거 |
| 선택지 분기 | 거의 제거 |
| 평판/팩션 | 제거 |
| 퀘스트 추적 UI | 단순 구현 |
| 퀘스트 보상 | 명확하게 구현 |
| 업적 파밍 | 플레이타임 계산에서 제외 |

### 권장 퀘스트 볼륨

| 종류 | 개수 | 역할 |
|---|---:|---|
| 메인 퀘스트 | 8~12개 | 엔딩까지 밀어주는 핵심 진행 |
| 서브 퀘스트 | 6~10개 | 탐색/파밍 동기 부여 |
| 숨겨진 미니 퀘스트 | 2~3개 | 발견 보상, 재방문 이유 |
| 반복 퀘스트 | 0개 | 볼륨 뻥튀기 느낌 방지 |

추천 총량: **메인 10개 + 서브 8개 + 히든 3개** 정도.

### 목표 타입

| 목표 타입 | 예시 |
|---|---|
| 방문 | 폐쇄된 검문소에 도착 |
| 획득 | 의료 키트 3개 회수 |
| 처치 | 변이체 5마리 제거 |
| 상호작용 | 발전기 작동 |
| 운반/반납 | 연구 샘플 가져오기 |
| 탈출/귀환 | 기지로 복귀 |
| 보스 처치 | 지역 원흉 제거 |

### 퀘스트 데이터 필드

- 제목
- 설명
- 현재 목표
- 목표 타입
- 목표 수량
- 완료 조건
- 보상
- 선행 퀘스트
- 완료 후 대사/로그

### 좋은 퀘스트 밀도 설계

한 번 나갈 때 여러 목표가 겹치게 만든다.

예시:

- 메인: 통신탑 전원 복구
- 서브: 실종자 배낭 회수
- 서브: 군용 상자 2개 조사
- 파밍: SMG 부품 찾기
- 선택: 위험 구역 진입

이렇게 하면 플레이타임이 억지로 늘어난 느낌 없이 자연스럽게 확보된다.
