# 게임 기준

이 문서는 TunaSweeper에서 공통으로 사용하는 방향, 거리, 단위, 재화 같은 게임 기준을 기록한다.

## 방향 기준

- 월드 기준 북쪽은 `+X` 축이다.
- 화면 기준 12시 방향은 북쪽으로 본다.
- 따라서 화면 기준 12시 방향으로 이동하거나 배치한다고 말하면 월드 좌표에서는 `+X` 방향을 의미한다.
- `+Y` 축은 북쪽 기준 오른쪽 방향으로 본다.
- `+Z` 축은 위쪽 방향이다.

## 거리 기준

- Unreal Engine 기본 단위는 cm이다.
- 게임 내 `1m`는 Unreal 월드 좌표 `100` 단위로 본다.
- 예: 중심에서 북쪽으로 `7m`는 중심 기준 `+X 700` 위치를 의미한다.
