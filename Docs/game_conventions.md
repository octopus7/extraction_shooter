# 게임 기준

이 문서는 TunaSweeper에서 공통으로 사용하는 방향, 거리, 단위, 재화 같은 게임 기준을 기록한다.

## 방향 기준

- 월드 기준 북쪽은 `+X` 축이다.
- 화면 기준 12시 방향은 북쪽으로 본다.
- 따라서 화면 기준 12시 방향으로 이동하거나 배치한다고 말하면 월드 좌표에서는 `+X` 방향을 의미한다.
- `+Y` 축은 북쪽 기준 오른쪽 방향으로 본다.
- `+Z` 축은 위쪽 방향이다.

## 거리 기준

- Unreal Engine 기본 단위는 cm이다.
- 게임 내 `1m`는 Unreal 월드 좌표 `100` 단위로 본다.
- 예: 중심에서 북쪽으로 `7m`는 중심 기준 `+X 700` 위치를 의미한다.

## 아이템 아이콘 기준

- 게임에서 사용하는 개별 아이템 아이콘 파일은 `256x256` 크기를 기준으로 한다.
- 4x4 아이템 아이콘 시트는 `1024x1024` 크기를 기준으로 하며, 각 칸은 `256x256`이다.
- 고해상도 작업 원본을 별도로 만들더라도 최종 게임용 개별 아이콘은 `256x256`으로 맞춘다.

## 시간 연출 기준

- 기본 페이드 연출 단위는 약 `0.2초`로 짧게 잡는다.
- 특별히 느린 전환이 필요한 경우가 아니면 UI/화면 전환 페이드는 `0.2초`를 기준으로 한다.
